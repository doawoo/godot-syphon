# Syphon Server/Client for Godot 4.5+ (Metal Backend Only)

This GDNative plugin that allows macOS Godot applications to utilize [Syphon](https://syphon.info)

See the `demo/` folder for working examples.

### Compiling

Ensure you clone this repo recursively, then, run SCons to build the plugin:

```
git clone --recursive https://github.com/doawoo/godot-syphon.git
cd godot-syphon
scons
```
